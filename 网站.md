# SEO优化

TDK三大标签：title、description、keyword

### title

​		不可替代，网页搜索入口主题归属

​		`网站名-网站介绍（尽量不超过30字）`

### description

​		简要说明网站主要做什么

### keyword

​		页面关键词，是搜索引擎关注点之一
